LocalDate firstday = LocalDate.of(today.getYear(),today.getMonth(),1);

PDF打印

设置打印副本（份数）

PrintRequestAttributeSet pras = new HashPrintRequestAttributeSet();
pras.add(new Copies(4));
pras.add(MultipleDocumentHandling.SEPARATE_DOCUMENTS_UNCOLLATED_COPIES);

注意：Copies和MultipleDocumentHandling需要配合使用，否则无效。

参考资料：https://docs.oracle.com/javase/6/docs/api/javax/print/attribute/standard/Copies.html

设置打印范围

pras.add(new PageRanges(4, 8));

单单设置这个打印熟悉是不够的，还需要搭配Book

PDDocument document = PDDocument.load(inputStream);
PDFPrintableHelper printable = new PDFPrintableHelper(document, printHandler);
PrinterJob job = PrinterJob.getPrinterJob();

job.setPrintService(defaultService);
PageFormat pageFormat = new PageFormat();

Paper paper = new Paper();
pageFormat.setPaper(paper);
Book book = new Book();

int pages = document.getNumberOfPages();
for (int pageNum = 1; pageNum <= pages; pageNum ++) {
    book.append(printable, pageFormat, pageNum);
}
job.setPageable(book);

若只book.append(printable, pageFormat);不管PDF有多少页都只打印第一页

重点：

1、在Printerable打印之前，它无法确定可能打印的页数

2、需要使用Pagable，这允许您收集一系列Printable，每个Printable代表虚拟书中的单个页面。

Mybatis

在Mybatis中，有两种占位符

  • #{}解析传递进来的参数数据
  • ${}对传递进来的参数原样拼接在SQL中
  • #{}是预编译处理(自动进行java类型和jdbc类型转换)，${}是字符串替换(不进行jdbc类型转换)。
  • 使用#{}可以有效的防止SQL注入，提高系统安全性。

当实体类中的属性名和表中的字段名不一样的解决方案

1、通过在查询的sql语句中定义字段名的别名，让字段名的别名和实体类的属性名一致

<select id=”selectorder” parametertype=”int” resultetype=”me.gacl.domain.order”>
    select order_id id, order_no orderno ,order_price price form orders where order_id=#{id};
</select>

2、通过来映射字段名和实体类属性名的一一对应的关系

<select id="getOrder" parameterType="int" resultMap="orderresultmap">
    select * from orders where order_id=#{id}
</select>
<resultMap type=”me.gacl.domain.order” id=”orderresultmap”>
    <!–用id属性来映射主键字段–>
    <id property=”id” column=”order_id”>
    <!–用result属性来映射非主键字段，property为实体类属性名，column为数据表中的属性–>
    <result property = “orderno” column =”order_no”/>
    <result property=”price” column=”order_price” />
</reslutMap>
  • Dao接口，就是人们常说的Mapper接口，接口的全限名，就是映射文件中的namespace的值，接口的方法名，就是映射文件中MappedStatement的id值，接口方法内的参数，就是传递给sql的参数。
